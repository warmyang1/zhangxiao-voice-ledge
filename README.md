# 张小账 - 智能语音记账APP

一款功能强大的语音记账应用，支持语音识别自动分类、AI消费分析、多平台消费同步。

## 功能特点

- 🎤 **语音记账** - 说话即可记录消费，自动智能分类
- 📊 **消费分析** - AI智能分析消费行为，洞察财务状况
- 🔗 **多平台同步** - 支持微信、淘宝、京东、抖音消费同步
- 💳 **订阅管理** - 灵活的周/月/年订阅方案
- 🔐 **安全可靠** - 本地数据存储，保护隐私

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- TailwindCSS 样式
- Zustand 状态管理
- Capacitor 跨平台打包

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```
